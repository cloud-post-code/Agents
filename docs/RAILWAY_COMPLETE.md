```python
def _build_database_url() -> str:
    """Auto-detects Railway PostgreSQL or uses localhost."""
    host = os.environ.get("PGHOST", "localhost")
    port = os.environ.get("PGPORT", "5433")  # Local dev
    user = os.environ.get("PGUSER", "postgres")
    password = os.environ.get("PGPASSWORD", "postgres")
    database = os.environ.get("PGDATABASE", "artisan")
    
    url = f"postgresql+asyncpg://{user}:{password}@{host}:{port}/{database}"
    
    # Add SSL for Railway
    if "railway.app" in host:
        url += "?sslmode=require"
    
    return url

database_url: str = _build_database_url()
```

**✅ Smart Detection:**
- Local dev: Uses `localhost:5433`
- Railway: Auto-detects `PGHOST`, `PGPORT`, etc.
- SSL: Automatically adds `?sslmode=require` for Railway

## Railway Setup (5 Minutes)

### 1. Add PostgreSQL Database

```
Railway Dashboard → New → Database → Add PostgreSQL
```

Railway automatically sets: `PGHOST`, `PGPORT`, `PGUSER`, `PGPASSWORD`, `PGDATABASE`

### 2. Set Application Variables

Railway Dashboard → Backend Service → Variables:

```bash
APP_ENV=production
DEBUG=false
SECRET_KEY=<generate-with: openssl rand -hex 32>
CORS_ORIGINS=https://your-frontend.railway.app
OPENAI_API_KEY=sk-your-key
```

### 3. Deploy

```bash
git push origin main
```

Railway auto-deploys ✓

### 4. Verify

```bash
curl https://your-backend.railway.app/api/v1/health
```

Expected:
```json
{"status": "healthy", "database": "connected"}
```

## Environment Variables Reference

### Railway Auto-Provides (PostgreSQL)
- `PGHOST` - Database host
- `PGPORT` - Database port (5432)
- `PGUSER` - Username (postgres)
- `PGPASSWORD` - Password
- `PGDATABASE` - Database name (railway)

### You Must Set Manually
- `APP_ENV` - Set to `production`
- `DEBUG` - Set to `false`
- `SECRET_KEY` - Generate secure random key
- `CORS_ORIGINS` - Your frontend URL
- `OPENAI_API_KEY` - OpenAI API key

### Optional
- `REDIS_URL` - Auto-set if you add Redis service

## Troubleshooting

### Connection Refused
**Issue**: Can't connect to PostgreSQL
**Solution**: Verify `PGHOST` is set in Railway variables

### SSL Required
**Issue**: SSL connection error
**Solution**: Already handled! Config auto-adds `?sslmode=require`

### CORS Error
**Issue**: Frontend can't connect
**Solution**: Update `CORS_ORIGINS` to exact frontend URL
